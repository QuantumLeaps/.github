# State Machines & Tools for Embedded Systems
We're on a mission to bring [Quantum Leaps](https://www.state-machine.com) in quality and productivity to embedded software development. Our open source [QP™ real-time embedded frameworks (RTEFs)](https://www.state-machine.com/products/qp), the freeware [QM™ modeling tool](https://www.state-machine.com/products/qm), and the [QTools™ collection](https://www.state-machine.com/products/qtools) enable event-driven architecture, finite state machines, model-based design and automatic code generation.
