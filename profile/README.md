# State Machines & Tools for Embedded Systems
[Quantum Leaps'](https://www.state-machine.com) [QP™ real-time embedded frameworks (RTEFs)](https://www.state-machine.com/products/qp), the freeware [QM™ modeling tool](https://www.state-machine.com/products/qm), and the [QTools™ collection](https://www.state-machine.com/products/qtools) bring **quantum leaps** in quality and productivity to embedded software development. They enable embedded developers to effectively apply event-driven architecture, hierarchial state machines, model-based design and automatic code generation.
